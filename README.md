# Bussin

Clean rebuild of the Tsadie bus location and parent alert application.

## Applications

- `bussin-frontend`: parent and driver web experiences
- `bussin-server`: API, database access, live trip state, and notifications
- `bussin-shared`: truthful request/response contracts shared by both sides

The old Buswatch project is a behavioral reference only. Bussin does not import or depend on it.

## Commands

```bash
npm install
npm run check
npm run build
```
